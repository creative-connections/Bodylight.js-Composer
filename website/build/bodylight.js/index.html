Bodylight.js Composer

Composer is the main product of the Bodylight.js, it allows you to visually create an HTML simulator. We are using the wonderful project GrapesJS to provide the layouting engine, Adobe® Animate to create interactive animations and Modelica to make it all run on models your write yourself.
